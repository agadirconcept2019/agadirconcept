# Deprecated prototype artifacts

This `src/` folder contains early Sprint 0/1 prototype artifacts (schemas/content contracts) from a pre-stack-decision phase.

## Important
- It is **not** the official runtime implementation path.
- Official implementation stack is now **PHP 8+ / Laravel / MySQL** under Laravel-standard directories:
  - `app/`
  - `routes/`
  - `database/`
  - `resources/views/`

Python runtime files were deprecated/removed in favor of the Laravel implementation.
